# Monoprism

## Manual configuration

To make this work, it needs a Google Cloud Project ([console](https://console.cloud.google.com/)):
* "Google Drive API" added from Library, used to access the Drive files
* An "OAUth 2.0 Client ID", used to log in users for private access
* An "API key", used for logged-out public access

## Testing

For testing:
* Make sure the Google Account email is added under Google Auth Platform > Audience
